# 직렬화
데이터 구조나 객체를 전송 가능한 포멧(JSON, XML)로 변환하는 과정.  

# 역직렬화
직렬화의 반대 과정  
데이터를 원래의 객체 구조나 데이터 구조로 복구하는 과정을 의미.  

**사용 범위**
- 네트워크 통신
  - 클라이언트와 서버 간 통신에서 직렬화/역직렬화를 통해 데이터를 전송
- 데이터 저장
  - 데이터베이스 등에 직렬화를 통해 저장
  - 불러올 때 역직렬화로 복구함
- 캐싱
  - 메모리 캐싱에 직렬화, 캐싱된 메모리에서 불러올 때 역직렬화

## 자바에서
**Jackson** 을 사용한 직렬화
- 자바에서 DTO를 그대로 응답해주면
- 클라이언트 딴에서는 자동으로 JSON으로 받을 수 있도록 직렬화해줌
