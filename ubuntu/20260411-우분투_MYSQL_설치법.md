# 우분투 MySql 설치법

## 1. 설치
`sudo apt update`

`sudo apt install mysql-server`

## 2. 시스템 세팅
`sudo ufw allow mysql`

`sudo systemctl start mysql`

`sudo systemctl enable mysql`

## 3. root 계정 비밀번호 번경
- `sudo mysql -u root`
    - `-p` 붙이면 비밀번호 입력 가능
- `use mysql;`
    - mysql DB 사용
- `ALTER USER 'root'@'localhost' IDENTIFIED WITH caching_sha2_password BY '비밀번호';`
    - 그낭 바꿀 시 암호화가 되지 않은 체로 저장됨
- `FLUSH PRIVILEGES;`
    - 번경사항 저장
- `exit;`
