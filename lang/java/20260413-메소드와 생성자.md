# 메소드와 생성자

## 메소드란
```
접근제어자 반환타입 메소드이름(매개변수목록) { // 선언부
    // 구현부
}
```
- 접근 제어자 : 헤당 메소드에 접근할 수 있는 범위를 명시
- 반환 타입 : 메소드가 모든 작업을 마치고 반환하는 데이터의 타입을 명시
- 메소드 이름 : 메소드를 호출하기 위한 이름을 명시
- 메개변수 목록 : 메소드 매개변수들의 목록을 명시
- 구현부 : 메소드의 고유 기능

## 생성자
자바에서 객체의 생성과 동시에 인스턴스의 변수를 원하는 값으로 초기화하는 메서드
**생성자의 선언 방법**
- `클래스이름() {} // 매개변수 없음`
- `클래스이름(A, B, C ...) {} // 매개변수 있음`
`Car myCar = new Car("아반떼", 2016, "흰색", 200); // 생성자의 호출`

**모든 클래스에는 하나 이상의 생성자가 정의되어 있어야 하지만, 자바 컴파일러가 매개변수 없는 기본 생성자를 추가해줌**

## this, this()
- this
  - 해당 인터페이스가 자기 자신의 주소를 가리킴
  - `this.color = color` => 인스턴스의 color을 바꿈
- this()
  - 다른 생성자(매개변수가 맞는)를 호출함
  - `this("소나타", 2012, "검정색", 160); // 다른 생성자를 호출함.`

## 메소드 오버로딩
**같은 이름의 메소드를 매개변수의 종류와 수만 다르게 중복하여 정의하는 것**
```
1. println()
2. println(boolean x)
3. println(char x)
4. println(char[] x)
5. println(double x)
6. println(float x)
7. println(int x)
8. println(long x)
9. println(Object x)
10. println(String x)
```
