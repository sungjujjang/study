# ProxMox 네트워크 브리지 설정

### 기본 vlan0 브리지를 사용할 시
- 각각의 VM마다 공유기에서 다른 아이피를 할당받게 됨
- 공유기에 부하가 걸릴 수 있을 뿐더러, 포트포워딩이 어려움

### vlan1을 새로 만들어 사용하기
- VM끼리 따로 네트워크 대역을 묶음
- 호스트 서버 VM이 공유기로부터 아이피를 받고,\
  VM들에게 정해진 아이피 대역에서 아이피를 나누어 줌

**브리지만 만들면 각 VM에 \
아이피를 자동으로(DHCP) 할당시키지 않음**

→ 마스터 VM을 만든 뒤에 아이피를 할당시키는 서버로 만듦
- 마스터 VM에서는 브리지를 2개
  - vlan0, vlan1
- vlan0에서 할당받은 공유기 아이피를 가지고
- vlan1에 마스터 아이피(ex: 10.0.0,1)를 지정
- vlan1로 통신하는 모든 아이피는 마스터 VM이 할당받은 vlan0 아이피로 나감/들어감
