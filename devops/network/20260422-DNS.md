# DNS(Domain Name Server)
URL을 실제 아이피 주소로 매핑시키는 네임서버
- 1.1.1.1 (cloudflare)
- 8.8.8.8/8.8.4.4 (google)


## URL -> IP 변환 순서
-  URL 입력
1. /etc/host.conf 조회
  - 로컬에 저장된 URL인지 확인
  - 만일 저장되어 있다면 네임서버 호출 없이 그대로 전송
2. /etc/resolv.conf 조회
  - 저장된 네임서버에 질의
3. 네임서버 질의
  - 만일 네임서버에서 응답한다면, 그대로 전송
  - 아니면 응답 없음

## DNS 서버 구조
**DNS는 계층 구조이다**  
최상위에 루트 DNS 서버가 존재하고, 그 밑으로 연속된 계층 구조로 이어져 있다.
